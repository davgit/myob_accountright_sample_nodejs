AccountRight Live API

node.js sample

This sample will step you through connecting to and fetching data from the MYOB AccountRight Live API.

The setup steps are:

  • Download (or checkout) the project
  • Run npm install
  • Enter your api credentials
  • Run node app.js

Your api credentials

You can apply for and will receive API keys from the developers.myob.com website. Once you have these simply update the following items:

set MYOBAPI_SAMPLE_CLIENTKEY to your API Key
set MYOBAPI_SAMPLE_CLIENTSECRET to your API secret
